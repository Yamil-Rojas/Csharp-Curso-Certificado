## 📖 Descripción del Curso
Este proyecto recopila mis primeros pasos en el lenguaje C# y la lógica de programación. El objetivo es consolidar las bases del desarrollo de software (variables, condicionales, bucles y vectores) utilizando aplicaciones de consola en .NET para obtener la certificación del primer nivel.

## 📁 Estructura de Unidades
El repositorio se organiza según las guías prácticas del curso:
* `📁 /Ejercicios-Unidad-2`: Primeros programas, variables y operadores aritméticos.
* `📁 /Ejercicios-Unidad-3`: Estructuras condicionales simples y compuestas (`if`, `else`).
* `📁 /Ejercicios-Unidad-4`: Operadores lógicos (`&&`, `||`) y condiciones combinadas.
* `📁 /Ejercicios-Unidad-5`: Bucles y estructuras de repetición (`for`, `while`, `do-while`).
* `📁 /Ejercicios-Unidad-6`: Combinación de ciclos y estrategias de corte de control.
* `📁 /Ejercicios-Unidad-7`: Arreglos y vectores (almacenamiento de datos indexados).
* `📁 /Ejercicios-Unidad-8`: Funciones, modularización y paso de parámetros.

## 🛠️ Requisitos Previos
Para compilar y ejecutar estas soluciones necesitas:
1. [.NET SDK](https://microsoft.com) (Versión 8.0 o superior recomendada).
2. [Visual Studio](https://visualstudio.microsoft.com/) o [Visual Studio Code](https://visualstudio.com) con la extensión oficial de C#.

## 💻 Cómo Ejecutar los Proyectos
1. Clona el repositorio en tu máquina local:
   ```bash
   git clone https://github.com
   ```
2. Navega hasta la carpeta de la unidad y el ejercicio que quieras revisar:
   ```bash
   cd Ejercicios-Unidad-2/Ejercicio1
   ```
3. Ejecuta la aplicación de consola desde tu terminal:
   ```bash
   dotnet run
   ```
